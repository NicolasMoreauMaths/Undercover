🔧 Configuration Firebase
Pour jouer en ligne, tu dois créer un projet Firebase gratuit.
Suis ces étapes :
  1. Va sur console.firebase.google.com et crée un projet
  2. Dans Build → Realtime Database, crée une base de données (mode test)
  3. Dans Authentication → Sign-in method, active Anonymous
  4. Dans Project Settings → Your apps, clique ➕ Web, copie la config
  5. Colle les valeurs ci-dessous et clique Sauvegarder
